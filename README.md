# Работай Курьером — Лендинг

Современный лендинг для привлечения людей к регистрации курьером.

## Запуск

```bash
npm install
npm run dev
```

Сайт будет доступен по адресу: http://localhost:5173

## Сборка для продакшена

```bash
npm run build
npm run preview
```

## Технологии

- React 19
- Vite
- CSS (без внешних UI-библиотек)
